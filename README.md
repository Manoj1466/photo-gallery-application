Photo Gallery Application (MERN Stack)

Overview

This project is a Photo Gallery Application built using the MERN (MongoDB, Express.js, React, Node.js) stack. The application allows users to upload, view, and manage images efficiently with a user-friendly interface.

Features

User Authentication (Login & Signup)

Upload Photos with descriptions

View & Manage Images in a gallery format

Search & Filter images by category

Responsive Design for a seamless user experience

Technologies Used

Frontend: React, Redux, Bootstrap, Axios

Backend: Node.js, Express.js

Database: MongoDB (Mongoose)

Authentication: JWT (JSON Web Token), bcrypt.js

File Storage: Multer (for image uploads)

Installation

Prerequisites

Ensure you have the following installed:

Node.js

MongoDB
